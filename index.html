technology stack

I have been working with the three main web technologies below and I have some indept knowlege of how they work together to create powerful web applications and websites.

Hyper-Text Mark-Up Language
HTML is the mark-up language that is used to define the structure of the web page.

Cascading Style Sheet
CSS is a stylesheet that contains a set of rules which is used for styling the web page. Typically it defines the colour, fonts and some other aspact of what the user sees.

JavaScript is a programming language used to add interactivity and implements behaviour to a static web page built using pure HTML and CSS. It implements behaviours like clicks, animation and some more advanced features.

Adds on

As an add on to the three fundamental technologies for web, I came across some bunch of technologies and learn them on my own. I now as a self-taught developer, I can code full stack applications with database integration on the backend using these powerful libraries and frameworks.

Node.js is not a framework, library nor a programming language. It is a JavaScript run-time environment that uses Chrome's V8 engine to executes JavaScript code outside of the browser.

React.js is a JavaScript front-end library that is used to build the user interface (UI) of the website or web applications.

Express.js is a JavaScript back-end framework that is used to build the API endpoints that will communicate with the front-end like React-JS. This allows to build full stack web application using JavaScript with the help of Node-JS, whick is a runtime environment for executing JavaScript code on the server.

Flask is full stack Python lightweight web framework for building website and web applications.

Django is also another powerful full stack Python web framework for building website and web applications.
